# 🍽️ Savoria Restaurant

A modern, responsive restaurant website built with React + Vite. Features a full menu, cart system, and live order experience.

## 🛠️ Tech Stack

- **React 18** — UI library
- **Vite 5** — Lightning-fast build tool
- **React Router DOM v6** — Client-side routing (HashRouter)
- **Recharts** — Data visualization
- **React Icons** — Icon library
- **Context API** — Cart & data state management

## 🧑‍💻 Getting Started

### Prerequisites
- Node.js v18+
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/Aakuz-Lusion/FUTURE_FS_03.git

# Navigate to the project
cd FUTURE_FS_03

# Install dependencies
npm install

# Start dev server
npm run dev
```

App runs at `http://localhost:3000`

---

## 📦 Build & Deploy

```bash
# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```
